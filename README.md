# 落語

落語网站前端项目，此项目基于 [Ant Design Pro V5](https://pro.ant.design) 和 [UmiJs 4](https://umijs.org)

点击访问网站：[落語](https://to2.fun)

## 安装 `node_modules`

`npm install` or `yarn`

## 运行

```bash
npm start
```

## 打包

```bash
npm run build
```

## 检查代码样式

```bash
npm run lint
```

你可以使用脚本自动修复一些错误:

```bash
npm run lint:fix
```
